Validação da Evolution API
1. Verificar Conexão

Consulta o estado da instância WhatsApp na Evolution API (GET /instance/connectionState/{instancia}). A instância precisa estar com o QR Code escaneado para retornar open.

2. Enviar Mensagem de Teste

Envia uma mensagem real para o número informado, independente do Modo de Teste — para confirmar que a API está funcionando end-to-end.

Somente dígitos: DDD + número. Ex: 11999998888
Passo a passo para configurar a Evolution API
  1. Instale a Evolution API via Docker: docker run -d --name evolution-api -p 8080:8080 atendai/evolution-api:latest
  2. Acesse o painel em http://localhost:8080 e crie uma instância com o nome configurado acima (ex: pascoa).
  3. Escaneie o QR Code com o WhatsApp do número que fará os disparos.
  4. Clique em Verificar Conexão — deve retornar state: open.
  5. Clique em Enviar Teste com o seu próprio número para confirmar o envio end-to-end.
  6. Após validado: desmarque Modo de Teste no canal WhatsApp e salve.
